# find47-dashboard

「FIND/47」プロジェクトのダッシュボードアプリケーションです。都道府県別の「FIND/47」作品数を表示し、スライドショーで作品を閲覧できます。

## デモ
- https://code4fukui.github.io/find47-dashboard/
- https://code4fukui.github.io/find47-dashboard/keywoard.html

## 機能
- 都道府県ごとの「FIND/47」作品数の表示
- 都道府県別のスライドショー表示
- キーワード検索によるスライドショー表示

## データ
- https://find47.jp/ のオープンデータを使用しています。
